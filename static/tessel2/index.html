Chapter 4: Tessel 2

The Tessel 2 combines a system-on-chip with embedded Linux and a microcontroller to build IoT prototypes and products.

Tessel 2

One core feature of the Tessel 2, is its command line interface based on Node.js.

The Tessel 2 is based on a system-on-chip [Mips 24K], see the datasheet for more details.

With a Tessel 2, you can follow the Tessel 2 Start Experience.

You can find the Tessel 2 Documentation here.

The main code repository is located at Tessel Project Main Repo.

Tessel modules

Some examples of Tessel modules are:

  • Servo module
  • Relay module

You can easily design your own Tessel module with help of a CAD program like Eagle.

Once designed, you can order and share the board on sites like oshpark.com.
